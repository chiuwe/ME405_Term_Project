This constructor creates a task which controls the speed of a motor using input from an A/D converter run through a potentiometer as well as an input from. a button. The main job of this constructor is to call the constructor of parent class (frt_task ); the parent's constructor the work.

Parameters:
a_nameA character string which will be the name of this task
a_priorityThe priority at which this task will initially run (default: 0)
a_stack_sizeThe size of this task's stack in bytes (default: configMINIMAL_STACK_SIZE)
brake_maskMask for brake value.
p_driverPointer to a motor driver.
p_brakePointer to the shared brake boolean.
p_powerPointer to the shared power value.
p_potPointer to the shared potentiometer boolean.
adc_maskMask for which adc to read from.
p_ser_devPointer to a serial device (port, radio, SD card, etc.) which can be used by this task to communicate (default: NULL)

void task_motor::run ( void  )

This run method is called by the RTOS and contains a loop in which the task checks for data and sends it if appropriate.

This method is called once by the RTOS scheduler. Each time around the for (;;) loop, it reads the A/D converter and change the selected motors speed. Each loop checks if it is being controled by a pot or by a given value and reacts accordingly

void task_solenoid::run ( void  )

This run method is called by the RTOS and contains a loop in which the task checks for data and sends it if appropriate.

This method is called once by the RTOS scheduler. Each time around the for (;;) loop, it reads the A/D converter and change the selected motors speed. Each loop also check the two additional buttons, which control the brakes of the individual motors.
